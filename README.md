# 🎮 Snack, Water and Gun Game

A simple **Snack, Water, and Gun** game built using Python. This is a fun command-line game where the user competes against the computer. The computer randomly selects an option, and the winner is decided based on the game rules.

## 📌 About the Project

This project is a Python implementation of the classic **Snack, Water, and Gun** game. It demonstrates the use of:

- Variables
- Conditional statements (`if-elif-else`)
- Loops
- Random module
- User input
- Basic game logic

It is a great beginner-friendly project for learning Python programming.

---

## 🎯 Game Rules

The game has three choices:

- 🐍 **Snack**
- 💧 **Water**
- 🔫 **Gun**

### Winning Rules

- 🐍 Snack drinks 💧 Water → **Snack Wins**
- 💧 Water drowns 🔫 Gun → **Water Wins**
- 🔫 Gun kills 🐍 Snack → **Gun Wins**
- If both players choose the same option, the match is a **Draw**.
